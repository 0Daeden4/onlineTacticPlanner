Help

Controls:

  • Select tools from the Tools & Icons category.
  • Use the Select tool to select and manipulate objects.
  • Use the selection filters to choose what types of objects are selectable.
  • Use the Draw tool to draw on the canvas.
  • Adjust brush size and color using the provided controls.
  • Press 'x' to delete the selected object.
  • Right-click to cancel the current action.
  • Use Ctrl+Z to undo the last action.
  • Use Ctrl+Y to redo the last undone action.
  • Zoom in and out using the mouse wheel.
  • Pan the canvas by holding the middle mouse button and moving the mouse.
  • Use the Clear All button to remove all objects from the canvas.
  • Set a background image from the Utilities category.
  • Toggle Dark Theme from the Utilities category.
  • Set the Max Icon Size in the Icons section to cap icon dimensions.
